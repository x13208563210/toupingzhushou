直播投屏助手 Windows 接收端说明
版本：v0.4.12
日期：2026-04-29

启动方式
1. 直接运行 `直播投屏助手-v0.4.12-2026-04-29.exe`。
2. 首次运行会弹出管理员权限确认，请点“是”，这样才能读取视频号直播伴侣的“互动消息”悬浮窗。
3. 如果弹出 Windows 防火墙提示，请允许局域网访问。

主要功能
1. 接收安卓投屏画面和声音。
2. 支持语音控制和语音点歌。
3. 支持弹幕提醒、礼物提醒、弹幕播报、自动回复。
4. 支持虚拟摄像头输出，方便 OBS 等软件采集。

使用说明
1. 弹幕提醒页负责探测视频号直播助手的“互动消息”悬浮窗，并做弹幕提醒与播报。
2. 自动回复页可以重新记录输入框和发送按钮坐标，再配置规则。
3. 分享包里的自动回复坐标、规则和 WebView 缓存都已重置，到对方电脑后需要重新设置。

注意事项
1. `voice-runtime`、`voice-model`、`apple-airplay`、`web-ui` 等目录不要删除。
2. 如果弹幕识别不到，请先确认视频号直播助手的“互动消息”悬浮窗已打开并可见。
3. 接收端现在会自动请求管理员权限，避免重启后出现“直播伴侣是管理员、接收端不是管理员”导致的读不到弹幕。
4. 如果需要虚拟摄像头，请以管理员身份运行安装相关组件。
